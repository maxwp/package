Subject: Технические требования к серверу h2. PHP
PHP 5.2 или выше с поддержкой библиотек gd2, mbstring, iconv, mysql

Возможность для php выставить:
memory_limit=128M
max_post_size=50M
file_upload_size=25M
register_globals=Off
display_errors=On
time_limit = 600 (10 minutes)

h2. MySQL
MySQL 3 или выше с поддержкой кодировок UTF-8, windows-1251.

h2. Web-Server
Веб-сервер - любой, но PHP должен работать в режиме php-cgi или php-fcgi (тоесть, процессы php должны запускаться с под имени пользователя, а не с под юзера www, а также возможность юзеру самостоятельно менять свой php.ini).
Веб-сервер НЕ_должен по умолчанию выбрасывать заголовок content-type: text/html; encoding=windows-1251;
Веб-сервер должен понимать файлы .htaccess с директивами:

RewriteEngine (mod_rewrite)
deny from all
require valid-user (mod_auth)
Предпочтительный веб-сервер: apache 2.2

h2. OS
Операционная система сервера:
Любая, кроме Windows и кроме Fedora.
Предпочтительно: debian, freebsd, centos, ubuntu.
