Пакет Events - отдельная простая реализация событийной системы.
Сам по себе пакет Events не представляет какого-либо интереса. Events
разработан специально для встраивания в другие системы и пакеты.

Встраивание допускается путем наследования или аггрегации пакета внутри
другого класса.

Впервые Events стал использовать пакет Engine для организации внутри Engine
своей событийной системы.

===========================
Подключение в #wpp
===========================
PackageLoader::Get()->import('Events');

Зависимости: пакет не имеет зависимостей.

===========================
Design pattern пакета
===========================
Events сделан по паттерну Observer (он же Publish-Subscribe, Listener,
наблюдатель, слушатель и так далее).
